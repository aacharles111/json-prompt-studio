Pure compilation functions:
```typescript
// Convert 0-1 fractional coords to 0-1000 grid [y1,x1,y2,x2]
export function normBbox(el: { x: number; y: number; w: number; h: number }): [number, number, number, number]

// Cartesian product of variable elements, capped at 10 combos
export function expandVariables(elements: Element[]): Element[][]

// Assemble the full IdeogramCaption from state
export function assembleCaption(state: AppState): IdeogramCaption

// Minify JSON string (single-line, no spaces, preserve non-ASCII)
export function minifyCaption(caption: IdeogramCaption): string

// Full compile pipeline: expand → filter disabled → order by layer → normBbox if precision → assemble → minify → return string[]
export function compile(state: AppState): string[]
```
Implement these with complete logic. The normBbox: y1=round(y*1000), x1=round(x*1000), y2=round((y+h)*1000), x2=round((x+w)*1000), clamp 0-1000, ensure y1<y2 and x1<x2.
